# Aula 6 - Revisão de HTML e de CSS
### HTML
Linguagem de Marcação de HiperTexto (HiperText Markup Language). 

```html
<!DOCTYPE html>
<head>
    <title>Document</title>
</head>
<body>
    <!-- conteúdo da página-->
</body>
</html>
```
